# Todo-приложение (Планировщик задач)

## Описание проекта

Todo-приложение — это веб-сервер на Go с REST API для управления задачами. Приложение позволяет создавать, редактировать, удалять и отмечать задачи как выполненные. Поддерживается функциональность повторяющихся задач с различными правилами повторения.

## Инструкция по запуску

### Предварительные требования
- Go 1.23 или выше

### Установка и запуск

1. **Клонирование репозитория:**
```bash
git clone -b first-iteration https://github.com/cldsty/go_final_project.git
cd go_final_project
```

2. **Установка зависимостей:**
```bash
go mod download
```

3. **Запуск сервера:**
```bash
go run main.go
```

Сервер запустится на порту **7540**.

### Доступ к приложению

Откройте браузер и перейдите по адресу:
```
http://localhost:7540
```

### Настройки

Все настройки заданы в коде:
- Порт: 7540
- База данных: scheduler.db
